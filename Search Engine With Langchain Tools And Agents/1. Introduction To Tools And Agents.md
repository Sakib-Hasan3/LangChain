## 1. **Introduction to Tools and Agents (LangChain)** — বাংলা ব্যাখ্যা

এই topicটা LangChain-এর **core philosophy**।
এটা না বুঝে তুমি যতই code লেখো, তুমি শুধু API caller হয়ে থাকবে—AI system builder না।

---

# 🔷 1. Tools কী?

> **Tool = এমন capability যা LLM নিজে পারে না, কিন্তু call করে করতে পারে**

LLM নিজে:

* real-time data fetch করতে পারে না
* file delete করতে পারে না
* database access করতে পারে না
* accurate calculation সবসময় করতে পারে না

👉 তাই tools দরকার

---

## 🛠️ Examples of Tools

* calculator
* web search
* database query
* file operations
* API calls

---

## 🔷 2. Agent কী?

> **Agent = এমন system যা decide করে কোন tool কখন use করতে হবে**

এটা শুধু response generate করে না
👉 এটা **decision নেয়**

---

## 🔥 Brutal Truth

অনেকেই ভাবে:

> “LLM + tool = agent”

❌ ভুল

👉 Agent = LLM + tools + decision logic + execution loop

---

# 🔷 3. Tools vs Agent

| বিষয়    | Tool              | Agent                     |
| ------- | ----------------- | ------------------------- |
| কাজ     | specific function | decision maker            |
| control | passive           | active                    |
| example | calculator        | decides when to calculate |

---

# 🔷 4. কেন Tools + Agents দরকার?

কারণ pure LLM:

* hallucinate করে
* outdated knowledge
* action নিতে পারে না

👉 Tools + Agent system:

* real-world interaction করতে পারে
* dynamic task solve করতে পারে

---

# 🔷 5. Simple Flow

```text
User → Agent → Decide → Tool → Result → Final Answer
```

# 🔷 7. LangChain-এ কীভাবে কাজ করে?

### Step-by-step:

1. User input
2. LLM reasoning
3. Tool selection
4. Tool execution
5. Observation
6. Final response

---

# 🔷 8. Agent loop (critical)

```text
Thought → Action → Observation → Thought → Final Answer
```

👉 এটাকে ReAct pattern বলা হয়

---

# 🔷 9. Code-level Concept

### Tool

```python
@tool
def multiply(a: int, b: int) -> int:
    return a * b
```

### Agent

```python
agent = create_agent(llm, tools=[multiply])
```

# 🔷 11. Tools ছাড়া Agent = useless

Agent যদি tool use করতে না পারে:

* শুধু text generate করবে
* real কাজ করতে পারবে না

---

# 🔷 12. Agent ছাড়া Tools = dumb

Tools থাকলেও:

* কে decide করবে কোন tool লাগবে?
  👉 agent ছাড়া useless

# 🔷 13. Common Mistakes

## ❌ সবকিছু agent দিয়ে করা

---

## ❌ tool description weak

→ wrong tool selection

---

## ❌ control না রাখা

→ risky behavior

---

## ❌ debugging না করা

→ blind system
