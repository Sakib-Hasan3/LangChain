## 2. **Creating Tools Using LangChain** — বাংলা ব্যাখ্যা

এই topicটা অনেকেই হালকা ভাবে নেয়, আর এখানেই তাদের system দুর্বল হয়ে যায়।

কারণ:

> **Agent-এর quality ≠ model-এর quality
> Agent-এর quality = tool design-এর quality**

---

# 🔷 1. Tool তৈরি করা মানে আসলে কী?

> **Python function → structured, LLM-usable capability বানানো**

মানে:

* function থাকবে
* clear input থাকবে
* clear output থাকবে
* description থাকবে (LLM বুঝার জন্য)

---

# 🔥 Brutal Truth

তুমি যদি শুধু function লিখে ভাবো “tool বানিয়ে ফেলেছি” —
👉 তুমি নিজেকে deceive করছ

কারণ LLM tool use করবে **description বুঝে**, code না পড়ে

---

# 🔷 2. সবচেয়ে basic way: `@tool` decorator

এটাই LangChain-এর standard method

```python
from langchain_core.tools import tool

@tool
def multiply(a: int, b: int) -> int:
    """Multiply two numbers."""
    return a * b
```

---

## 🧠 এখানে কী হলো?

LangChain এই function থেকে extract করে:

* name → `multiply`
* description → `"Multiply two numbers."`
* inputs → `a`, `b`
* output → `int`

👉 এখন LLM এটা use করতে পারবে

---

# 🔷 3. Docstring = Brain of Tool

```python
@tool
def search_weather(city: str) -> str:
    """Get current weather for a given city."""
    return "Sunny"
```

👉 এই description না থাকলে:

LLM বুঝবেই না:

* কখন use করবে
* কেন use করবে

---

## ❌ Bad Example

```python
@tool
def do_something(x):
    """Does something."""
```

👉 completely useless

---

# 🔷 4. Tool Design Formula

একটা ভালো tool বানাতে এই ৪টা ঠিক রাখো:

| Element     | Rule                     |
| ----------- | ------------------------ |
| Name        | clear & specific         |
| Description | কখন use করবে explain করো |
| Input       | typed & structured       |
| Output      | clean & predictable      |

---

# 🔷 5. Multiple Tools Example

```python
from langchain_core.tools import tool

@tool
def add(a: int, b: int) -> int:
    """Add two integers."""
    return a + b

@tool
def multiply(a: int, b: int) -> int:
    """Multiply two integers."""
    return a * b

tools = [add, multiply]
```

👉 এখন agent decide করবে:

* add লাগবে নাকি multiply

---

# 🔷 6. Tool + LLM Integration

```python
from langchain_openai import ChatOpenAI
from langchain.agents import create_tool_calling_agent, AgentExecutor
from langchain_core.prompts import ChatPromptTemplate

llm = ChatOpenAI(model="gpt-4o-mini")

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant"),
    ("human", "{input}"),
    ("placeholder", "{agent_scratchpad}")
])

agent = create_tool_calling_agent(llm, tools, prompt)
executor = AgentExecutor(agent=agent, tools=tools, verbose=True)

response = executor.invoke({"input": "What is 5 * 7?"})
print(response)
```

# 🔷 9. Tool Input Schema

Typed input important:

```python
@tool
def divide(a: float, b: float) -> float:
    """Divide two numbers."""
    return a / b
```

👉 typed না হলে:

* LLM ভুল input দিতে পারে
* parsing fail হবে
