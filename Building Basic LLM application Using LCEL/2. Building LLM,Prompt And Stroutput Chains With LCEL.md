## Building LLM, Prompt & StrOutput Chains with LCEL

**LangChain**-এ LCEL (LangChain Expression Language) হলো modern way to build chains — clean, composable, production-friendly।

👉 পুরনো `LLMChain` style এখন outdated।
👉 LCEL = modular + readable + scalable pipeline।

---

# 🧠 LCEL কী?

LCEL হলো এমন syntax যেখানে তুমি **components pipe (`|`) দিয়ে connect করো**।

```python
prompt | llm | output_parser
```

এটাই chain।

---

# 🎯 এই project-এ কী বানাবো

আমরা ৩টা জিনিস combine করবো:

1. Prompt Template
2. LLM
3. String Output Parser

👉 Result: clean text output chain

# Cell 2 — Set API key

```python
import os
os.environ["OPENAI_API_KEY"] = "your_api_key_here"
```

### Explanation

LLM call করার জন্য API key লাগবে।

---

# Cell 3 — Import components

```python
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
```

### Explanation

* `ChatOpenAI` → LLM
* `ChatPromptTemplate` → structured prompt
* `StrOutputParser` → raw string output

---

# Cell 4 — Create the prompt

```python
prompt = ChatPromptTemplate.from_template(
    "Explain {topic} in simple terms."
)
```

### Explanation

এখানে dynamic prompt বানানো হচ্ছে।

Example:

```python
topic = "retrievers"
```

---

# Cell 5 — Create the LLM

```python
llm = ChatOpenAI(
    model="gpt-4.1-mini",
    temperature=0
)
```

### Explanation

* deterministic output
* learning phase-এ best

---

# Cell 6 — Create output parser

```python
parser = StrOutputParser()
```

### Explanation

LLM output structured object দেয় → parser এটাকে clean string বানায়।

---

# Cell 7 — Build LCEL chain

```python
chain = prompt | llm | parser
```

### Explanation

এটাই LCEL-এর core:

```text
Input → Prompt → LLM → Output Parser → Final Output
```

---

# Cell 8 — Run the chain

```python
result = chain.invoke({"topic": "LangChain"})
print(result)
```

### Explanation

* input dict format
* `{topic}` replace হবে
* final clean string return করবে

---

# Cell 9 — Try multiple inputs

```python
topics = ["retrievers", "chains", "embeddings"]

for t in topics:
    print("\nTopic:", t)
    print(chain.invoke({"topic": t}))
```

### Explanation

Same chain → multiple inputs
Reusable pipeline

---

# Cell 10 — Add system instruction (upgrade)

```python
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a strict teacher. Explain clearly in 2 sentences."),
    ("human", "Explain {topic}.")
])

chain = prompt | llm | parser

print(chain.invoke({"topic": "vector stores"}))
```

### Explanation

এখন behavior control হচ্ছে।

👉 LCEL advantage: easy modification

---

# Cell 11 — Add formatting constraint

```python
prompt = ChatPromptTemplate.from_template(
    "Explain {topic} in 3 bullet points."
)

chain = prompt | llm | parser

print(chain.invoke({"topic": "retrievers"}))
```

### Explanation

Prompt → output shape control করছে
