## ৯. LLM Structured Output — TypedDict দিয়ে

Pydantic শিখেছ। এবার দেখব **TypedDict** — Python-এর built-in একটি হালকা বিকল্প। কোনো extra লাইব্রেরি লাগে না, শুধু `typing` মডিউল থেকে import করলেই হয়।### TypedDict কী?

`TypedDict` হলো Python-এর `typing` মডিউলের একটি টুল, যেটা দিয়ে একটি dictionary-এর key আর তাদের type আগে থেকে define করে রাখা যায়। এটা Pydantic-এর মতো runtime-এ data validate করে না — বরং code editor আর type checker (যেমন `mypy`) কে বলে দেয় "এই dict-এ কোন key থাকবে, কোন type থাকবে।"

---

### ধাপ ১ — TypedDict দিয়ে Schema বানাও

```python
from typing import TypedDict, Optional

class Book(TypedDict):
    title: str
    author: str
    year: int
    genre: Optional[str]
```

**আউটপুট** (এটি শুধু type definition, কোনো object তৈরি হয় না — কিন্তু manually বানালে দেখাবে):

```python
book: Book = {
    "title": "নন্দিত নরকে",
    "author": "হুমায়ূন আহমেদ",
    "year": 1972,
    "genre": "উপন্যাস"
}
print(book)
```
```
{'title': 'নন্দিত নরকে', 'author': 'হুমায়ূন আহমেদ', 'year': 1972, 'genre': 'উপন্যাস'}
```

### ধাপ ২ — OpenAI-এর `parse()` দিয়ে সরাসরি ব্যবহার

OpenAI SDK (v1.50+) TypedDict সরাসরি `response_format` হিসেবে নেয়:

```python
from openai import OpenAI
from typing import TypedDict

client = OpenAI()

class Book(TypedDict):
    title: str
    author: str
    year: int

response = client.beta.chat.completions.parse(
    model="gpt-4o-mini",
    response_format=Book,
    messages=[
        {"role": "user", "content": "হুমায়ূন আহমেদের একটি বিখ্যাত বইয়ের তথ্য দাও"}
    ]
)

book = response.choices[0].message.parsed
print(book)
print(type(book))
print(book["title"])
print(book["year"])
```

**আউটপুট:**
```
{'title': 'মিসির আলি অমনিবাস', 'author': 'হুমায়ূন আহমেদ', 'year': 1987}
<class 'dict'>
মিসির আলি অমনিবাস
1987
```

### ধাপ ৩ — Nested TypedDict

```python
from typing import TypedDict, List

class Author(TypedDict):
    name: str
    birth_year: int
    nationality: str

class Book(TypedDict):
    title: str
    author: Author      # nested TypedDict
    year: int
    tags: List[str]

response = client.beta.chat.completions.parse(
    model="gpt-4o-mini",
    response_format=Book,
    messages=[
        {"role": "user", "content": "রবীন্দ্রনাথের গীতাঞ্জলি বইয়ের বিস্তারিত তথ্য দাও"}
    ]
)

book = response.choices[0].message.parsed
print(book)
print("বইয়ের নাম:", book["title"])
print("লেখক:", book["author"]["name"])
print("জন্মসাল:", book["author"]["birth_year"])
print("ট্যাগ:", book["tags"])
```

**আউটপুট:**
```
{
  'title': 'গীতাঞ্জলি',
  'author': {
      'name': 'রবীন্দ্রনাথ ঠাকুর',
      'birth_year': 1861,
      'nationality': 'ভারতীয়'
  },
  'year': 1910,
  'tags': ['কবিতা', 'নোবেল পুরস্কার', 'ভক্তিমূলক']
}

বইয়ের নাম: গীতাঞ্জলি
লেখক: রবীন্দ্রনাথ ঠাকুর
জন্মসাল: 1861
ট্যাগ: ['কবিতা', 'নোবেল পুরস্কার', 'ভক্তিমূলক']
```

### ধাপ ৪ — List of TypedDict (একাধিক item)

```python
from typing import TypedDict, List

class Book(TypedDict):
    title: str
    author: str
    year: int

class BookList(TypedDict):
    books: List[Book]
    total: int

response = client.beta.chat.completions.parse(
    model="gpt-4o-mini",
    response_format=BookList,
    messages=[
        {"role": "user", "content": "৩টি বিখ্যাত বাংলা উপন্যাসের তথ্য দাও"}
    ]
)

result = response.choices[0].message.parsed
print(f"মোট বই: {result['total']}")
for b in result["books"]:
    print(f"  - {b['title']} ({b['year']}) — {b['author']}")
```

**আউটপুট:**
```
মোট বই: 3
  - পদ্মা নদীর মাঝি (1936) — মানিক বন্দ্যোপাধ্যায়
  - আরণ্যক (1939) — বিভূতিভূষণ বন্দ্যোপাধ্যায়
  - পথের পাঁচালী (1929) — বিভূতিভূষণ বন্দ্যোপাধ্যায়
```

### TypedDict বনাম Pydantic — কখন কোনটা?

| বিষয় | TypedDict | Pydantic |
|---|---|---|
| লাইব্রেরি | শুধু `typing` (built-in) | `pip install pydantic` |
| Runtime validation | নেই | আছে |
| ভুল type হলে | চুপ থাকে | Error দেয় |
| OpenAI SDK support | সরাসরি | instructor দিয়ে |
| ব্যবহারের উপযুক্ততা | ছোট প্রজেক্ট, দ্রুত prototype | Production, critical data |

সহজ কথায়: data নিয়ে কঠোর নিয়ম দরকার হলে **Pydantic**, আর হালকা ও দ্রুত কিছু বানাতে চাইলে **TypedDict** — দুটোই LLM structured output-এর জন্য চমৎকার।
