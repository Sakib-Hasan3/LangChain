## 7. **Message Types in LangChain** — বাংলা ব্যাখ্যা

এই topicটা surface-level মনে হলেও আসলে এটা **LangChain-এর communication protocol**।
তুমি যদি এটা ঠিকমতো না বোঝো, তাহলে:

* chat history ভুল হবে
* agent ভুল behave করবে
* tool calling break করবে

---

# 🔷 1. Message Types কী?

LangChain-এ **messages** হলো structured way-তে conversation represent করার method।

👉 LLM-এ raw string না দিয়ে structured message পাঠানো হয়

---

## 🧠 Core Idea

> **Conversation = list of messages**

```python
messages = [
    {"role": "system", "content": "..."},
    {"role": "user", "content": "..."},
    {"role": "assistant", "content": "..."}
]
```

LangChain এটাকে class-based করে manage করে।

---

# 🔷 2. Main Message Types

LangChain-এ সবচেয়ে important message types:

---

## 🧑‍💻 1. HumanMessage

👉 user input represent করে

```python
from langchain_core.messages import HumanMessage

HumanMessage(content="What is LangChain?")
```

---

## 🤖 2. AIMessage

👉 model-এর response

```python
from langchain_core.messages import AIMessage

AIMessage(content="LangChain is a framework...")
```

---

## ⚙️ 3. SystemMessage

👉 model-এর behavior control করে

```python
from langchain_core.messages import SystemMessage

SystemMessage(content="You are a helpful assistant")
```

---

## 🛠️ 4. ToolMessage

👉 tool execution result represent করে

```python
from langchain_core.messages import ToolMessage

ToolMessage(content="Result is 42", tool_call_id="abc123")
```

---

## 🔁 5. FunctionMessage (older / legacy)

👉 tool-এর পুরনো version concept

---

# 🔥 Brutal Truth

যদি তুমি শুধু string pass করো:

```python
llm.invoke("Explain AI")
```

👉 তুমি LangChain properly use করছ না

কারণ:

* no context
* no role separation
* no control

---

# 🔷 3. Message Flow (Critical)

একটা full conversation flow:

```text
SystemMessage → rules
HumanMessage → question
AIMessage → response
ToolMessage → tool result
AIMessage → final answer
```

---

# 🔷 4. Practical Example

```python
from langchain_core.messages import HumanMessage, SystemMessage
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(model="gpt-4o-mini")

messages = [
    SystemMessage(content="You are a helpful assistant"),
    HumanMessage(content="Explain LangChain")
]

response = llm.invoke(messages)
print(response.content)
```

---

## 🧠 এখানে কী হলো?

* system → behavior define করলো
* human → question দিল
* model → structured input পেল
* output → AIMessage হিসেবে আসলো

---

# 🔷 5. Message vs Prompt

👉 অনেকেই confuse হয়

| Prompt        | Messages                |
| ------------- | ----------------------- |
| single string | structured conversation |
| static        | dynamic                 |
| less control  | more control            |

---

# 🔷 6. ChatPromptTemplate

LangChain message automate করতে use করে:

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are helpful"),
    ("human", "{question}")
])
```

---

# 🔷 7. Tool Calling + Messages

Agent use করলে message আরও critical হয়ে যায়

Flow:

1. HumanMessage → input
2. AIMessage → tool call suggestion
3. ToolMessage → tool result
4. AIMessage → final answer

---

## ⚠️ Important

ToolMessage ছাড়া agent loop break হবে

### ❌ system message ignore করা

---

### ❌ tool message না বোঝা

→ agent broken

---

### ❌ message order ভুল
