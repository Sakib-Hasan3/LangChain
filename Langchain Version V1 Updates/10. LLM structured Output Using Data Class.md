## ১০. LLM Structured Output — Data Class দিয়ে### Data Class কী?

Python-এর `@dataclass` decorator দিয়ে এমন একটি class বানানো যায় যেটা automatically `__init__`, `__repr__`, `__eq__` তৈরি করে দেয়। TypedDict-এর মতো built-in — কোনো extra লাইব্রেরি লাগে না। তবে TypedDict থেকে পার্থক্য হলো এটা একটা সত্যিকারের **Python object** দেয়, dictionary নয়। তাই `book["title"]` না লিখে `book.title` লেখা যায়।

---

### ধাপ ১ — @dataclass দিয়ে Schema বানাও

```python
from dataclasses import dataclass
from typing import Optional

@dataclass
class Book:
    title: str
    author: str
    year: int
    genre: Optional[str] = None
```

**আউটপুট** (manually object বানালে):

```python
book = Book(title="পথের পাঁচালী", author="বিভূতিভূষণ", year=1929)
print(book)
print(book.title)
print(book.year)
```
```
Book(title='পথের পাঁচালী', author='বিভূতিভূষণ', year=1929, genre=None)
পথের পাঁচালী
1929
```

### ধাপ ২ — OpenAI `parse()` দিয়ে LLM Call

```python
from dataclasses import dataclass
from typing import Optional
from openai import OpenAI

client = OpenAI()

@dataclass
class Book:
    title: str
    author: str
    year: int
    genre: Optional[str] = None

response = client.beta.chat.completions.parse(
    model="gpt-4o-mini",
    response_format=Book,
    messages=[
        {
            "role": "user",
            "content": "হুমায়ূন আহমেদের একটি বিখ্যাত বইয়ের তথ্য দাও"
        }
    ]
)

book = response.choices[0].message.parsed

print(book)
print("নাম   :", book.title)
print("লেখক :", book.author)
print("সাল  :", book.year)
print("ধরন  :", book.genre)
```

**আউটপুট:**
```
Book(title='নন্দিত নরকে', author='হুমায়ূন আহমেদ', year=1972, genre='উপন্যাস')
নাম   : নন্দিত নরকে
লেখক : হুমায়ূন আহমেদ
সাল  : 1972
ধরন  : উপন্যাস
```

### ধাপ ৩ — Nested Data Class

```python
from dataclasses import dataclass
from typing import List

@dataclass
class Author:
    name: str
    birth_year: int
    nationality: str

@dataclass
class Book:
    title: str
    author: Author        # nested dataclass
    year: int
    tags: List[str]

response = client.beta.chat.completions.parse(
    model="gpt-4o-mini",
    response_format=Book,
    messages=[
        {
            "role": "user",
            "content": "রবীন্দ্রনাথের গীতাঞ্জলি বইয়ের বিস্তারিত তথ্য দাও"
        }
    ]
)

book = response.choices[0].message.parsed

print(book)
print("বইয়ের নাম  :", book.title)
print("লেখকের নাম :", book.author.name)
print("লেখকের জন্ম:", book.author.birth_year)
print("জাতীয়তা   :", book.author.nationality)
print("ট্যাগসমূহ  :", book.tags)
```

**আউটপুট:**
```
Book(
  title='গীতাঞ্জলি',
  author=Author(name='রবীন্দ্রনাথ ঠাকুর', birth_year=1861, nationality='ভারতীয়'),
  year=1910,
  tags=['কবিতা', 'আধ্যাত্মিকতা', 'নোবেল পুরস্কার']
)
বইয়ের নাম  : গীতাঞ্জলি
লেখকের নাম : রবীন্দ্রনাথ ঠাকুর
লেখকের জন্ম: 1861
জাতীয়তা   : ভারতীয়
ট্যাগসমূহ  : ['কবিতা', 'আধ্যাত্মিকতা', 'নোবেল পুরস্কার']
```

### ধাপ ৪ — List of Data Class

```python
from dataclasses import dataclass
from typing import List

@dataclass
class Book:
    title: str
    author: str
    year: int

@dataclass
class BookList:
    books: List[Book]
    total: int

response = client.beta.chat.completions.parse(
    model="gpt-4o-mini",
    response_format=BookList,
    messages=[
        {
            "role": "user",
            "content": "৩টি বিখ্যাত বাংলা উপন্যাসের তথ্য দাও"
        }
    ]
)

result = response.choices[0].message.parsed

print(f"মোট বই: {result.total}\n")
for b in result.books:
    print(f"  {b.title} ({b.year}) — {b.author}")
```

**আউটপুট:**
```
মোট বই: 3

  পদ্মা নদীর মাঝি (1936) — মানিক বন্দ্যোপাধ্যায়
  আরণ্যক (1939) — বিভূতিভূষণ বন্দ্যোপাধ্যায়
  পথের পাঁচালী (1929) — বিভূতিভূষণ বন্দ্যোপাধ্যায়
```

### ধাপ ৫ — Data Class-এ Custom Method যোগ করা

Data Class-এর বিশেষ সুবিধা হলো এতে নিজের method লেখা যায় — TypedDict বা Pydantic-এ এটা এত সহজ নয়।

```python
from dataclasses import dataclass
from typing import List

@dataclass
class Book:
    title: str
    author: str
    year: int
    pages: int

    def age(self) -> int:
        return 2025 - self.year

    def summary(self) -> str:
        return f'"{self.title}" — {self.author}, {self.year} সালে প্রকাশিত ({self.pages} পৃষ্ঠা)'

    def is_classic(self) -> bool:
        return self.age() > 50

response = client.beta.chat.completions.parse(
    model="gpt-4o-mini",
    response_format=Book,
    messages=[
        {
            "role": "user",
            "content": "পথের পাঁচালী বইয়ের তথ্য দাও — পৃষ্ঠা সংখ্যাসহ"
        }
    ]
)

book = response.choices[0].message.parsed

print(book.summary())
print("বয়স (বছর)  :", book.age())
print("ক্লাসিক কি? :", book.is_classic())
```

**আউটপুট:**
```
"পথের পাঁচালী" — বিভূতিভূষণ বন্দ্যোপাধ্যায়, 1929 সালে প্রকাশিত (352 পৃষ্ঠা)
বয়স (বছর)  : 96
ক্লাসিক কি? : True
```

### তিনটি পদ্ধতির চূড়ান্ত তুলনা

| বিষয় | Pydantic | TypedDict | Data Class |
|---|---|---|---|
| লাইব্রেরি | `pip install pydantic` | built-in | built-in |
| Runtime validation | আছে | নেই | নেই |
| Access style | `obj.field` | `obj["key"]` | `obj.field` |
| Custom method | সম্ভব | সম্ভব নয় | সহজে সম্ভব |
| OOP সুবিধা | আংশিক | নেই | পূর্ণ |
| উপযুক্ত ক্ষেত্র | Production, critical data | দ্রুত prototype | OOP প্রজেক্ট |

সহজ কথায় — **Data Class** হলো TypedDict আর Pydantic-এর মাঝামাঝি একটি চমৎকার বিকল্প। এটা built-in, dot notation সাপোর্ট করে, আর custom method লেখা যায় — তাই OOP-style Python প্রজেক্টে এটি খুবই মানানসই।
