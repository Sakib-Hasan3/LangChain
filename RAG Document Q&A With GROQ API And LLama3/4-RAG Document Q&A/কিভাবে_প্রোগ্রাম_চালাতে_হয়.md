# RAG ডকুমেন্ট Q&A প্রোগ্রাম চালানোর নির্দেশিকা

## প্রোগ্রাম সম্পর্কে
এই প্রোগ্রামটি একটি **RAG (Retrieval-Augmented Generation)** ভিত্তিক প্রশ্নোত্তর সিস্টেম যা PDF ডকুমেন্ট থেকে তথ্য সংগ্রহ করে এবং GROQ API এর মাধ্যমে Llama3 মডেল ব্যবহার করে উত্তর প্রদান করে।

---

## দুটি ফাইলের পার্থক্য

### 1. **app_huggingfaceembedding.py** 
এই ফাইলটি **HuggingFace Embeddings** ব্যবহার করে। এটি ব্যবহার করতে OpenAI API কী প্রয়োজন নেই।

### 2. **main.py**
এই ফাইলটি **OpenAI Embeddings** ব্যবহার করে। এটি ব্যবহার করতে OpenAI API কী প্রয়োজন।

---

## প্রয়োজনীয় API কী সেটআপ

### প্রয়োজনীয় কী:
1. **GROQ_API_KEY** - GROQ API থেকে (বাধ্যতামূলক)
2. **HF_TOKEN** - HuggingFace টোকেন (শুধুমাত্র `app_huggingfaceembedding.py` এর জন্য)
3. **OPENAI_API_KEY** - OpenAI API কী (শুধুমাত্র `main.py` এর জন্য)

### `.env` ফাইল তৈরি করুন:

প্রোগ্রাম ডিরেক্টরিতে একটি `.env` ফাইল তৈরি করুন এবং নিম্নলিখিত কন্টেন্ট যোগ করুন:

```
GROQ_API_KEY=আপনার_GROQ_API_কী_এখানে_রাখুন
OPENAI_API_KEY=আপনার_OPENAI_API_কী_এখানে_রাখুন
HF_TOKEN=আপনার_HUGGINGFACE_টোকেন_এখানে_রাখুন
```

## ধাপ ৫: ওয়েব ইন্টারফেস ব্যবহার করুন

Streamlit অটোমেটিক্যালি আপনার ব্রাউজারে খুলবে (`http://localhost:8501`)

### প্রোগ্রাম ব্যবহারের ধাপ:

1. **"Document Embedding" বাটন ক্লিক করুন** - এটি আপনার PDF ডকুমেন্টগুলি থেকে ভেক্টর এম্বেডিং তৈরি করবে এবং FAISS ডাটাবেসে সংরক্ষণ করবে।

2. **প্রশ্ন লিখুন** - প্রদত্ত টেক্সট বক্সে আপনার প্রশ্ন লিখুন।

3. **ফলাফল দেখুন** - প্রোগ্রাম নিম্নলিখিত দেখাবে:
   - LLM এর কাছ থেকে উত্তর
   - "Document Similarity Search" বিভাগে প্রাসঙ্গিক ডকুমেন্ট অংশ

```
PDF ডকুমেন্ট লোড

         ↓
FAISS ভেক্টর স্টোরে সংরক্ষণ
         ↓
ব্যবহারকারীর প্রশ্ন নিন
         ↓
প্রাসঙ্গিক ডকুমেন্ট খুঁজুন (Retriever)
         ↓
Llama3-8b-8192 মডেলে পাঠান
         ↓
উত্তর প্রদান করুন
```

## সাধারণ সমস্যা এবং সমাধান

### ১. `.env` ফাইল পড়া হচ্ছে না
- নিশ্চিত করুন যে `.env` ফাইল প্রোগ্রাম ফোল্ডারে আছে
- `python-dotenv` প্যাকেজ ইনস্টল করেছেন কিনা যাচাই করুন

### ২. API কী অবৈধ বলে ত্রুটি আসছে
- GROQ কনসোল থেকে নতুন API কী তৈরি করুন
- ফাইলে সঠিক কী পেস্ট করেছেন কিনা যাচাই করুন

### ৩. PDF লোড হচ্ছে না
- `research_papers` ফোল্ডার আছে কিনা চেক করুন
- PDF ফাইলগুলি এই ফোল্ডারে আছে কিনা নিশ্চিত করুন

### ৪. Streamlit বন্ধ হয়ে যাচ্ছে
```bash
streamlit run app_huggingfaceembedding.py --logger.level=debug
```
এভাবে চালান বিস্তারিত ত্রুটি দেখতে।

### ৫. মেমরি সমস্যা
কম ডকুমেন্ট দিয়ে শুরু করুন এবং ধীরে ধীরে বাড়ান।

- **chunk_size**: ১০০০ থেকে বৃদ্ধি করুন দ্রুত প্রসেসিংয়ের জন্য
