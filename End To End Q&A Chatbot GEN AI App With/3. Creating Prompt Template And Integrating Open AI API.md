## 3. Creating Prompt Template and Integrating OpenAI API — বাংলা ব্যাখ্যা

---

## 1) Prompt Template কী?

Prompt template হলো এমন একটা reusable pattern, যেখানে কিছু অংশ fixed থাকে, আর কিছু অংশ variable থাকে। LangChain-এ এটা prompt-কে dynamic, reusable, এবং structured করে। ([LangChain Docs][1])

সহজ উদাহরণ:

* fixed অংশ: “তুমি একজন সহায়ক AI assistant”
* variable অংশ: `{topic}`

মানে, একই structure বারবার ব্যবহার করা যাবে, শুধু input বদলাবে।

---

## 2) কেন Prompt Template দরকার?

তুমি raw string লিখে কাজ চালাতে পারো, কিন্তু সেটা beginner-level habit।

Prompt template দরকার কারণ:

* prompt reuse করা যায়
* input variables cleanly inject করা যায়
* chain-এর সাথে সহজে জোড়া লাগে
* system/human message আলাদা রাখা যায় ([LangChain Docs][2])

Brutal truth:
যদি তুমি prompt template ছাড়া random string ছুঁড়তে থাকো, তুমি system design করছ না, trial-and-error করছ।

---

## 3) LangChain-এ Prompt Template কিভাবে বানায়?

### Simple `PromptTemplate`

এটা single-string prompt-এর জন্য।

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template(
    "Explain {topic} in simple terms."
)

formatted_prompt = prompt.format(topic="LangChain")
print(formatted_prompt)
```

এখানে `{topic}` হলো variable।

---

## 4) Chat app হলে `ChatPromptTemplate` বেশি useful

কারণ modern chat model-গুলো message-based input ভালোভাবে handle করে। LangChain chat models, যেমন `ChatOpenAI`, message-oriented interface ব্যবহার করে। ([LangChain Docs][3])

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "Explain {topic} in Bangla.")
])
```

এখন `{topic}` dynamically replace হবে।

---

## 5) OpenAI API integration মানে কী?

OpenAI API integration মানে তুমি OpenAI model-কে programmatically call করছ। OpenAI-এর Python SDK দিয়ে API key ব্যবহার করে model request পাঠানো যায়, আর OpenAI-এর primary modern text-generation interface হলো Responses API। ([OpenAI Developers][4])

LangChain-এর দিক থেকে, OpenAI model use করতে সাধারণত `langchain-openai` package install করে `ChatOpenAI` wrapper ব্যবহার করা হয়। ([LangChain Docs][3])

---

## 6) OpenAI API use করতে কী কী লাগে?

তিনটা জিনিস:

* OpenAI account
* API key
* required Python packages

OpenAI quickstart অনুযায়ী API key create করে environment variable হিসেবে `OPENAI_API_KEY` set করা হয়। LangChain OpenAI integration docs-ও `langchain-openai` install আর API key setup-এর কথা বলে। ([OpenAI Developers][4])

## 9) LangChain + OpenAI basic integration

এটাই core pattern:

```python
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="gpt-5.4",
    temperature=0
)

response = llm.invoke("Explain LangChain simply.")
print(response.content)
```

LangChain docs অনুযায়ী `ChatOpenAI` হলো OpenAI chat model integration wrapper, আর OpenAI docs-এ `gpt-5.4`-এর মতো models Responses API examples-এ দেখানো হয়েছে। ([LangChain Docs][3])

## 10) এবার Prompt Template + OpenAI একসাথে

এটাই topic-এর আসল জিনিস।

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "Explain {topic} in Bangla.")
])

llm = ChatOpenAI(
    model="gpt-5.4",
    temperature=0.3
)

chain = prompt | llm

response = chain.invoke({"topic": "Prompt Template"})
print(response.content)
```

### এখানে কী হচ্ছে?

* `prompt` = reusable instruction structure
* `llm` = OpenAI model connection
* `prompt | llm` = LCEL chain
* `invoke({...})` = variable inject করে model run করা

LangChain overview-এ model integrations এবং pipe-based composition pattern LangChain-এর central workflow হিসেবে দেখানো হয়েছে। ([LangChain Docs][1])

## 11) OpenAI official SDK দিয়ে direct call কেমন হয়?

LangChain ছাড়া pure SDK pattern roughly এমন:

```python
import os
from openai import OpenAI

client = OpenAI(api_key=os.environ.get("OPENAI_API_KEY"))

response = client.responses.create(
    model="gpt-5.4",
    input="Explain LangChain in simple terms."
)

print(response.output_text)
```

OpenAI Python library docs অনুযায়ী Python SDK-এর primary interface Responses API, এবং `client.responses.create(...)` হলো standard pattern। ([OpenAI Developers][6])

### Difference:

* **OpenAI SDK** = direct API usage
* **LangChain** = prompting, chaining, tools, memory, agents-এর orchestration

## 12) কেন `ChatPromptTemplate` বেশি important?

কারণ chat systems-এ roles matter করে:

* system
* human
* assistant

এই role separation model behavior control করতে সাহায্য করে। LangChain chat model docs message-based structure-এর উপর জোর দেয়। ([LangChain Docs][3])

উদাহরণ:

```python
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are an expert Python teacher."),
    ("human", "Teach me {topic} with an example.")
])
```

এখানে system role model-এর behavior set করছে।
এটা ignore করলে output random হতে শুরু করবে।

## 13) Common mistakes

### 1. Prompt আর data একসাথে গুলিয়ে ফেলা

ভুল:

```python
llm.invoke("You are helpful. Explain AI. Also explain in Bangla. Also with example...")
```

এটা messy।

সঠিক approach:

* instruction আলাদা
* variable input আলাদা

---

### 2. API key code-এ hardcode করা

এটা sloppy এবং unsafe। OpenAI docs environment variable-based setup দেখায়। ([OpenAI Developers][4])

---

### 3. Wrong package install করা

অনেকে `langchain` install করে ভাবে সব হয়ে গেছে।
না। OpenAI wrapper-এর জন্য `langchain-openai` লাগে। ([LangChain Docs][3])

---

### 4. Model name blindly copy করা

সব model সব account-এ available নাও হতে পারে; OpenAI API-তে current model availability `/models` endpoint দিয়ে দেখা যায়। ([OpenAI Developers][7])

---

### 5. Prompt template না ব্যবহার করে copy-paste coding

এতে scaling impossible হয়ে যায়।

## 14) Full beginner-ready example

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# Prompt template
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful AI teacher."),
    ("human", "Explain {topic} in Bangla with a simple example.")
])

# OpenAI model
llm = ChatOpenAI(
    model="gpt-5.4",
    temperature=0.2
)

# Chain
chain = prompt | llm

# Run
response = chain.invoke({"topic": "virtual environment"})
print(response.content)
```
